Math's Got Your Back — PWA

Una micro PWA, offline-ready, che mostra il motto “Math's got your back” con una semplice animazione orbitale. Include toggle IT/EN, condivisione, install prompt, e un service worker per la cache.

Uso

  1. Metti tutti i file in una cartella su un hosting statico (GitHub Pages, Netlify, ecc.).
  2. Apri index.html. Installa come app dal browser.
